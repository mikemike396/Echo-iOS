# Echo-iOS
Simple RSS reader

## Tools
- Xcode version 15.0+

## Setup
- We use Firebase Realtime Datebase to drive the feed search index. To build this project locally you need to setup a personal Firebase project and include the plist at `Echo/GoogleService-Info.plist`.
